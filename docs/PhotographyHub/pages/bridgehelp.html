Update Master-steps.txt from Piper NAS to SandDisk

# Login to Piper NAS

  • Click on File Station
  • Click Transfers
  • Select MASTER-steps.txt
  • Right click on MASTER-steps.txt
  • Click Download
  • Use Finder and go to Downloads and select MASTER-steps.txt
  • Open another Finder window
  • Navigate to Extreme SSD / Backup_Files_To_NAS / PhotographyHub / pages
  • Drag MASTER-steps.txt from Downloads to Pages (choose replace)

# Invoke Bridge Tools

  • On tools, double click Build & Open Workflow Hub
  • Close Bridge Tools
  • Invoke Bridge Tools
